# MySepolia Sample Polygon NFT Contract

## Prerequisites

- [Node.js](https://nodejs.org/en/) (>= 10.16.0)

## Deployment

1. Install project

```bash
npm i
```

2. Create an `.env` and set `CONTRACT_ADDRESS` there

3. Paste your contract's ABI to `src/abi/msNFT.json`

3. Run with

```bash
npm start
```
